# Jiu Jitsu

## Sobre o Projeto

Este projeto tem como objetivo desenvolver um **sistema de gestão e controle de acesso para uma academia de Jiu-Jitsu**, modernizando a forma como alunos, professores e administradores interagem com o ambiente.  

Atualmente, a gestão da academia é feita de forma manual, especialmente no que se refere ao **controle de frequência, organização de aulas, evolução de faixas e registro de alunos**. Esse processo é pouco eficiente, propenso a erros e dificulta a tomada de decisões estratégicas.  

A proposta é implementar uma solução integrada que utilize **tecnologias modernas**, como **reconhecimento facial com o controlador Intelbras SS 3542 MF W**, além de um sistema web intuitivo para facilitar o acesso a informações relevantes.  

---


## Objetivos do Projeto  

- Automatizar o **controle de acesso e presença** dos alunos e professores.  
- Fornecer relatórios claros sobre **frequência, desempenho e evolução de faixas**.  
- Integrar **frontend, backend, banco de dados e nuvem** em uma solução única.  
- Criar uma interface simples e acessível para usuários com pouca familiaridade com tecnologia.  

---

## Público-Alvo  

O sistema beneficiará diretamente:  

1. **Alunos** – crianças, adolescentes e adultos interessados na prática e evolução no Jiu-Jitsu.  
2. **Professores** – instrutores responsáveis pelas aulas.  
3. **Gestores da academia** – responsáveis por administração, relatórios e organização interna.
